Introduction
------------

ArrayLookup is a fast array lookup library.

Features
--------

- [x] Search at least data, `once()`, `twice()`, `times()`


Installation
------------

** Require this library uses [composer](https://getcomposer.org/). **

```sh
composer require samsonasik/array-utils
```
